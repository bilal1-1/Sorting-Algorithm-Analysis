This document explains how artificial intelligence (AI) tools are used in project development, the scope of use, limitations, and validation processes.

# Artificial Intelligence Tools Used

ChatGPT(OpenAI GPT-4) and Gemini(2.5/3.0) models
Used for: troubleshooting suggestions, generating test data, optimization suggestions.

# Scope of Artificial Intelligence Support

AI tools have only been used in the following areas:
    -Creating texts for some chart libraries (JFreeChart etc.).
    -Suggestions for error detection.
    -Explanation of the working logic of some algorithms (Bubble Sort, Merge Sort etc.).
    -Editing of English-Turkish document texts.

Not used for:
    -The AI ​​did not directly write the entire project, the main code structure, or the underlying algorithms.
    -No code is produced without human control.

# Human Verification

All AI-generated content has been:
    -Proven, edited, and approved by the developer.
    -Tested in Java.
    -Checked for performance and accuracy.
No direct or uncontrolled use of AI output.

# Ethical Principles

The use of artificial intelligence in this project was conducted in accordance with:
    -The course's academic integrity principles,
    -IEEE ethical guidelines,
    -General software development standards.
The use of AI is transparently disclosed in this document.
